# NoteGenius AI Study Assistant

An AI-powered full-stack web application that converts PDF study notes into summaries, important topics, MCQs, and long-answer questions.

## Features

- Upload PDF notes
- Extract text from PDF
- Generate AI-based summaries
- Important topic extraction
- MCQ generation
- Long-answer question generation
- Clean dark-themed UI

## Tech Stack

### Frontend
- React.js
- Axios
- Vite

### Backend
- Node.js
- Express.js
- Multer
- pdf-parse

## Project Flow

PDF Upload → Text Extraction → AI Processing → Notes Generation
